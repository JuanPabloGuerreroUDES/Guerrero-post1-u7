## Funcionalidades disponibles
1. Ver lista de productos
Al ingresar a la URL, se muestra una tabla con los productos registrados.
Si no hay productos, se mostrará un mensaje indicando que no existen registros.
2. Crear un nuevo producto
Hacer clic en el botón "+ Nuevo Producto".
Completar el formulario:
Nombre
Descripción
Precio
Presionar el botón "Crear Producto".
El sistema redirige automáticamente a la lista actualizada.
3. Editar un producto
En la lista, hacer clic en el botón "Editar" de un producto.
Modificar los datos en el formulario.
Presionar el botón "Editar Producto".
Se regresa a la lista con los cambios aplicados.
4. Eliminar un producto
Hacer clic en el botón "Eliminar" de un producto.
Confirmar la acción en la ventana emergente.
El producto se elimina y la lista se actualiza automáticamente.
